# Challenge 01

- Progama que captura números informados pelo usuário, adiciona em uma lista e faz operações como soma, média, maior número menor número.

## Linguagem

- Java

## Funcionalidades 

- Menu de opções
- Captura de maior número
- Captura de menor número
- Consulta de soma dos números
- Consulta de média dos números

### Extras

- Tratamento para listas vazias.
